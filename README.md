# selectel-task

Тестовое задание для Selectel.

## Установка зависимостей

```
$ npm install
```

## Запуск сервера

```
$ npm start
```

Доступ к приложению после запуска сервера можно получить по адресу [`http://localhost:8080`](http://localhost:8080).

## Сборка проекта

```
$ npm run build
```

Результат будет находиться в папка `dist`.
